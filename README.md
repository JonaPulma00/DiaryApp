# DiaryApp - ASP.NET Core MVC

Personal diary application built with ASP.NET Core MVC and Entity Framework Core.

## Features
- Create, read, update, and delete diary entries
- Responsive dark theme UI with Tailwind CSS
- Basic log for development
- Implementation of ViewModel / DTO
- ASP Net Core Identity
- SQL Server database with EF Core migrations
- Clean MVC architecture

## Tech Stack
- ASP.NET Core 8.0
- Entity Framework Core
- SQL Server
- Tailwind CSS

## Setup
1. Update connection string in appsettings.json
2. Run migrations: `dotnet ef database update`
3. Run: `dotnet run`
